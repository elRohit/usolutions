# includes/

Aquest directori conté fitxers PHP reutilitzables i comuns per a tota la web:

- `db_connection.php`: Gestiona la connexió a la base de dades.
- `functions.php`: Conté funcions útils globals com la gestió de permisos, execució de scripts SSH, etc.
- `header.php` i `footer.php`: Elements comuns que es carreguen a totes les pàgines per mantenir coherència visual.
